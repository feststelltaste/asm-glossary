# Hallucination

> When an agent generates plausible-looking but incorrect code, e.g., calling non-existent APIs, misinterpreting business logic, or inventing dependencies. The primary risk in agent-assisted modernization, because hallucinated code can compile and appear to work on the surface. Test harnesses and code knowledge graphs are the most important countermeasures.
