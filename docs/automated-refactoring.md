# Automated Refactoring

> Structural code changes (Extract Method, Rename, Move Class) that an agent performs rule-based or LLM-assisted, in contrast to manual IDE refactorings. The advantage lies in scaling: an agent can consistently execute hundreds of similar refactorings across an entire codebase. A prerequisite is a reliable test harness that verifies behavior preservation after each change.
