# Sub Agent

> An AI agent that operates within a larger multi-agent system, executing a single specialized subtask as directed by an orchestrator. A sub agent receives a scoped context, performs its work, and returns a result. It has no awareness of the broader goal. This isolation keeps each agent focused and reduces the risk of one agent's mistakes cascading through the whole system.
