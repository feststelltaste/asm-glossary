# Attribution

> Attribution is the practice of explicitly linking each part of an agent's output to the specific source artifact it was derived from, such as a file, function, or document. Where grounding ensures an agent uses verified sources, attribution makes the connection visible and auditable, so reviewers can verify the reasoning behind generated or transformed code.
