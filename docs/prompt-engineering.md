# Prompt Engineering

> The deliberate crafting of instructions for the agent, including codebase context, architectural rules, and the desired target state. Particularly demanding in the modernization context, because the prompt must convey legacy-specific knowledge that is documented nowhere. Good prompt engineering partially compensates for the limits of the context window.
