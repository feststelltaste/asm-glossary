# RAG (Retrieval-Augmented Generation)

> A technique where the agent retrieves relevant context from the codebase, documentation, or architectural decisions before generating code. Overcomes context window limitations by loading only the currently relevant information. In the modernization context, often used in combination with a code knowledge graph.
