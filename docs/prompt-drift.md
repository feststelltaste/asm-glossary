# Prompt Drift

> The gradual divergence of an agent's behavior from the original intent over the course of a multi-turn session. As instructions accumulate and context shifts, the agent may silently reinterpret goals, adopt inconsistent conventions, or lose track of earlier constraints. Prompt drift is a key driver of context rot and can be mitigated through periodic re-anchoring to the original task specification.
