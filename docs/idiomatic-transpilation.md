# Idiomatic Transpilation

> Translating source code from one programming language to another by parsing it into an abstract syntax tree and generating code that follows the conventions, patterns, and idioms of the target language. Unlike mechanical transpilation, which produces a literal translation, idiomatic transpilation aims for output that reads as if it were written natively in the target language.
