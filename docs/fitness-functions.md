# Fitness Functions

> Automated, measurable checks that continuously monitor whether a system meets defined architectural and quality goals, e.g., modularity, performance budgets, dependency rules, or coupling metrics. Originating from the concept of evolutionary architecture, they give the agent objective criteria for whether a modernization change moves the system in the right direction. Without fitness functions, an agent optimizes blindly. With them, it has a measurable objective function.
