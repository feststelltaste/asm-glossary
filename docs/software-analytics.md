# Software Analytics

> The systematic extraction of actionable insights from software artifacts such as version histories, dependency graphs, runtime logs, and code metrics. Software analytics transforms raw repository and system data into signals that guide modernization decisions: identifying hotspots, measuring coupling, detecting architectural erosion, and quantifying technical debt. It provides the empirical foundation that makes Guided AI possible by supplying the data agents need to prioritize where and what to modernize.
