# Strangler Fig Pattern

> Incremental replacement of a legacy system with new implementations while the old system keeps running. Particularly well-suited for agent-assisted automation, because individual migration steps are isolated and repeatable. The agent can migrate feature by feature while the overall system remains functional at all times.
