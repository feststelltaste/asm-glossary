# Workload Design

> Defining the shape and boundaries of each work item in a modernization initiative, including whether it is performed by an agent, a human, other tools, or a combination of these. Workload design turns a high-level modernization goal into concrete, assignable units with clear ownership, tooling requirements, and success criteria.
