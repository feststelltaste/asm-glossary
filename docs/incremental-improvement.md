# Incremental Improvement

> The practice of modernizing legacy systems through many small, reversible changes rather than one big-bang rewrite. Each step leaves the system in a working state and can be verified independently. Incremental improvement is a natural fit for agents: shorter steps mean smaller context, smaller diffs, faster feedback from tests, and easier human review. It also contains the blast radius of any mistake the agent makes, keeping human oversight tractable.
