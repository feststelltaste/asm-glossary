# Agent Runbook

> A predefined recipe that scopes an agent's work to a single, well-bounded operation. An agent runbook removes ambiguity by spelling out exactly what the agent should do, which tools it may use, and how success is verified. Multiple runbooks can be composed to cover larger transformation efforts.
