# Agent Memory

> An agent's ability to retain and access information across steps and sessions. Memory ranges from the active context window, through persisted files such as agent instructions, plans, and specifications that are loaded at the start of a session, to external stores like vector databases or code knowledge graphs that are queried on demand. The right combination depends on the scope and duration of the work.
