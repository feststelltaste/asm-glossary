# Context Rot

> The progressive loss of accuracy and coherence in an agent's working context as a modernization session grows longer. Accumulated edits, superseded decisions, and obsolete code references silently degrade the agent's situational awareness, increasing the risk of hallucinations and inconsistent transformations. Techniques like summarization checkpoints, semantic anchors, and session segmentation help counteract this effect.
