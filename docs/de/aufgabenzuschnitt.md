# Aufgabenzuschnitt

> Das Festlegen von Form und Grenzen jedes Arbeitspaketes in einer Modernisierungsinitiative, einschließlich der Frage, ob es von einem Agenten, einem Menschen, anderen Tools oder einer Kombination davon ausgeführt wird. Workload Design überführt ein übergeordnetes Modernisierungsziel in eine Menge von Modernisierungsfällen, jeweils eine konkrete, zuweisbare Einheit mit klarer Verantwortlichkeit, Tooling-Anforderungen und Erfolgskriterien.
