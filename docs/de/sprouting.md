# Sprouting

![Sprouting](assets/images/sprouting.png)

> Eine Legacy-Code-Technik, bei der neue Funktionalität neben ungetestetem Legacy-Code entsteht, statt in ihn hineingewoben zu werden. Die neue Logik ist sauber, getestet und wird an einer kleinen, sorgfältig geprüften Stelle aus dem alten Code heraus aufgerufen. In der agentengestützten Modernisierung gibt Sprouting Agenten einen sicheren Ort, um modernen, gut getesteten Code zu schreiben, ohne jede umliegende Zeile vollständig verstehen oder transformieren zu müssen. Der unberührte Legacy-Code bleibt dabei eine bekannte Größe, während neues Verhalten isoliert heranwächst.
