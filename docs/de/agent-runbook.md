# Agent Runbook

> Ein vordefiniertes Rezept, das die Arbeit eines Agenten auf eine einzelne, klar abgegrenzte Operation beschränkt. Ein Agent Runbook beseitigt Mehrdeutigkeit, indem es genau festlegt, was der Agent tun soll, welche Tools er verwenden darf und wie der Erfolg überprüft wird. Mehrere Runbooks können kombiniert werden, um größere Transformationsvorhaben abzudecken.
