# Quellennachweis

> Quellennachweis ist die Praxis, jeden Teil der Ausgabe eines Agenten explizit mit dem spezifischen Quellartefakt zu verknüpfen, aus dem er abgeleitet wurde, etwa einer Datei, einer Funktion oder einem Dokument. Während Verankerung sicherstellt, dass ein Agent verifizierte Quellen verwendet, macht der Quellennachweis diese Verbindung sichtbar und nachvollziehbar, sodass Reviewer die Begründung hinter generiertem oder transformiertem Code überprüfen können.
