# MCP (Model Context Protocol)

> Standardisiertes Protokoll, über das Agents externe Tools und Datenquellen anbinden, ohne für jedes Tool eigene Integrationen bauen zu müssen. Ermöglicht eine einheitliche Schnittstelle zu IDEs, Build-Systemen, Datenbanken, Versionsverwaltung und weiteren Werkzeugen. Reduziert den Integrationsaufwand im Harness Engineering erheblich.
