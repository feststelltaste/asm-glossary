# Musterbasierter Vergleich

> Eine Änderungsdisziplin, bei der Modifikationen so gestaltet werden, dass sie über mehrere Dateien hinweg strukturell identische Diffs erzeugen, sodass Abweichungen sofort auffallen. Die Aufgabe des Reviewers verschiebt sich vom Verstehen jeder einzelnen Änderung hin zur Prüfung, ob das Muster eingehalten wird, und zur Untersuchung jeder Abweichung, sei es per Auge oder per automatisierter Ausreißererkennung über die Form der Diffs. Macht die Menge zum Vorteil: Je einheitlicher die Diffs, desto leichter lässt sich erkennen, welcher nicht dazugehört.
