# Mehrschrittplanung

> Die Fähigkeit eines Agents, ein komplexes Ziel in eine Folge konkreter Teilaufgaben zu zerlegen und diese Schritt für Schritt auszuführen, ohne dass ein Mensch jeden Schritt explizit vorgeben muss. Die Qualität der Planung bestimmt, wie weit ein Agent autonom arbeiten kann, bevor er korrigiert werden muss. Schlechte Planung führt zu kumulierenden Fehlern; gute Planung macht den agentischen Zyklus bei längeren Aufgaben produktiv.
