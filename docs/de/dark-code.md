# Dark Code

> Dark Code ist Produktionscode, den kein Mensch jemals in seinem gesamten Lebenszyklus vollständig verstanden hat. Anders als Legacy Code, der einst verstanden wurde, aber im Laufe der Zeit sein institutionelles Wissen verlor, wurde Dark Code generiert, hat automatisierte Prüfungen bestanden und wurde ausgeliefert, ohne dass der Schritt des Verstehens jemals stattfand. Er entsteht auf natürliche Weise in agentischen Workflows, in denen Urheberschaft und Verständnis entkoppelt sind.
