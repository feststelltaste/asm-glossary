# Fitnessfunktionen

> Automatisierte, messbare Prüfungen, die kontinuierlich überwachen, ob ein System definierte Architektur- und Qualitätsziele einhält, z.B. Modularität, Performance-Budgets, Abhängigkeitsregeln oder Kopplungsmetriken. Stammen aus dem Konzept der Evolutionary Architecture und liefern dem Agent objektive Kriterien, ob eine Modernisierungsänderung das System in die richtige Richtung bewegt. Ohne Fitnessfunktionen optimiert ein Agent blind; mit ihnen hat er eine messbare Zielfunktion.
