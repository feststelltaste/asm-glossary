# KI-Agent

> Ein LLM-basiertes System, das autonom mehrstufige Aufgaben ausführt, indem es plant, Tools aufruft und Ergebnisse evaluiert, und sich damit von reinem Prompt/Response unterscheidet. Im Modernisierungskontext bedeutet das: Der Agent analysiert Legacy-Code, leitet Refactoring-Schritte ab, führt sie durch und prüft das Ergebnis. Die Autonomie ist dabei bewusst begrenzt, denn ein Agent ohne Leitplanken ist ein Risiko, kein Werkzeug.
