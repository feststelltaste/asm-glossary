# Review-Müdigkeit

> Der Rückgang von Prüfqualität und Aufmerksamkeit, wenn Menschen große Mengen an agentengeneriertem Code in kurzer Folge validieren müssen. Wenn Agenten Änderungen schneller produzieren, als Reviewer sich vertieft damit auseinandersetzen können, verkommt der Review zum Abnicken, und subtile Fehler, Abweichungen von der Absicht oder unsichere Muster bleiben unentdeckt. Ein zentrales Risiko in der agentischen Modernisierung, weil die Ausgabemenge die kognitive Bandbreite der Menschen übersteigen kann.
