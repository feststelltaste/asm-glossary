# Idiomatische Transpilation

> Übersetzung von Quellcode aus einer Programmiersprache in eine andere, indem der Code in einen abstrakten Syntaxbaum überführt und daraus Code generiert wird, der den Konventionen, Mustern und Idiomen der Zielsprache folgt. Im Gegensatz zur mechanischen Transpilation, die eine wörtliche Übersetzung erzeugt, zielt idiomatische Transpilation auf ein Ergebnis ab, das sich liest, als wäre es nativ in der Zielsprache geschrieben worden.
