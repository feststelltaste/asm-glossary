# Kontextfenster

> Die maximale Menge an Text und Code, die ein LLM gleichzeitig verarbeiten kann. Bei Legacy-Systemen mit großen Codebasen ein kritischer Engpass, weil nie der gesamte Code gleichzeitig sichtbar ist. Strategien wie RAG, Code Knowledge Graphs und gezielte Kontextauswahl sind notwendig, damit der Agent trotzdem fundierte Entscheidungen trifft.
