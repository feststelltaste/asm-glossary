# Orchestrierung

> Koordination mehrerer Agents oder Agent-Schritte für komplexe Modernisierungsaufgaben. Beispiel: Ein Agent analysiert Abhängigkeiten, ein anderer refactored, ein dritter testet. Die Herausforderung liegt in der Übergabe von Kontext zwischen den Agents und der Sicherstellung, dass die Gesamtaufgabe konsistent bleibt.
