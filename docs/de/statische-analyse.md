# Statische Analyse

> Werkzeuge wie SonarQube, PMD oder ArchUnit, die ein Agent als Tool nutzt, um Codequalität, Abhängigkeiten und Architekturverletzungen zu prüfen. Liefert dem Agent objektive Metriken und strukturierte Informationen über die Codebasis. Im agentischen Zyklus dient die statische Analyse als Feedback-Kanal, der dem Agent hilft, die Auswirkungen seiner Änderungen zu bewerten.
