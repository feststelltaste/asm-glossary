# Software Analytics

> Die systematische Gewinnung verwertbarer Erkenntnisse aus Software-Artefakten wie Versionshistorien, Abhängigkeitsgraphen, Laufzeitprotokollen und Code-Metriken. Software Analytics wandelt rohe Repository- und Systemdaten in Signale um, die Modernisierungsentscheidungen leiten: Hotspots identifizieren, Kopplung messen, architektonische Erosion erkennen und technische Schulden quantifizieren. Es liefert das empirische Fundament, das Guided AI erst ermöglicht, indem es die Daten bereitstellt, die Agenten benötigen, um zu priorisieren, wo und was modernisiert werden soll.
