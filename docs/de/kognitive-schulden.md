# Kognitive Schulden

> Die schleichende Erosion des geteilten Systemverständnisses in einem Team über die Zeit. Kognitive Schulden leben in den Menschen: Sie entstehen, wenn Entwickler Code akzeptieren, den sie nicht vollständig verstehen, und wenn das Wissen darüber, wie ein System funktioniert und warum, auf zu wenige Köpfe verteilt ist. Besonders gefährlich, weil sie unsichtbar bleibt, bis sie sich als Entscheidungsunfähigkeit, Angst vor Änderungen oder unerwartetes Systemverhalten manifestiert. KI-generierter Code beschleunigt die Akkumulation massiv, weil der Lerneffekt des Selbst-Schreibens entfällt.
