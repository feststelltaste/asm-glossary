# Code-Wissensgraph

> Eine graphbasierte Repräsentation der Legacy-Codebasis und ihrer Zusammenhänge: Module, Klassen, Abhängigkeiten, Datenflüsse, Geschäftsregeln, Domänenkonzepte. Dient dem Agent als strukturiertes Langzeitgedächtnis, das über das Kontextfenster hinausgeht. Ermöglicht gezielte Abfragen wie „Welche Services hängen von diesem Entity ab?" statt den gesamten Code ins Kontextfenster laden zu müssen. Wird typischerweise im Rahmen des Harness Engineering aufgebaut.
