# Vibe Coding

> Ein Entwicklungsansatz, bei dem Entwickler ein KI-Modell im Gespräch anleiten, den generierten Code mit minimalem bis keinem Review akzeptieren und auf Basis davon iterieren, ob er funktioniert, statt ob er verstanden wird. Der Begriff wurde Anfang 2025 von Andrej Karpathy geprägt und fand schnell Eingang in den allgemeinen Sprachgebrauch. Während Vibe Coding für Prototyping und Exploration effektiv ist, beschleunigt es in Produktionsworkflows sowohl Dark Code als auch Kognitive Schulden.
