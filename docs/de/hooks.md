# Hooks

> Shell-Befehle oder Skripte, die ein Agent automatisch an definierten Lifecycle-Punkten ausführt, z.B. vor oder nach einem Tool-Aufruf, beim Sitzungsstart oder nach einer Dateiänderung. Ermöglichen die automatisierte Durchsetzung von Leitplanken, Formatierung, Logging oder Validierung, ohne explizite Agent-Anweisungen für jeden Aufruf. Im Modernisierungskontext verbinden Hooks den Agent mit der bestehenden Build- und Qualitätspipeline.
