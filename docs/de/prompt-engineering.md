# Prompt Engineering

> Das gezielte Formulieren von Anweisungen an den Agent, inklusive Codebasis-Kontext, Architekturregeln und gewünschtem Zielzustand. Im Modernisierungskontext besonders anspruchsvoll, weil der Prompt Legacy-spezifisches Wissen transportieren muss, das nirgends dokumentiert ist. Gutes Prompt Engineering kompensiert teilweise die Grenzen des Kontextfensters.
