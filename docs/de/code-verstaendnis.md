# Code-Verständnis

> Die Fähigkeit eines Agents, bestehenden Legacy-Code semantisch zu verstehen: Abhängigkeiten, Geschäftslogik, implizite Konventionen, gewachsene Strukturen. Geht über syntaktisches Parsing hinaus, da der Agent erkennen muss, warum Code so geschrieben wurde, nicht nur was er tut. Wird durch Code Knowledge Graphs und RAG unterstützt, um die Grenzen des Kontextfensters zu überwinden.
