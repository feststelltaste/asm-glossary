# Unteragent

> Ein KI-Agent, der innerhalb eines größeren Multi-Agent-Systems operiert und eine einzelne spezialisierte Teilaufgabe ausführt, wie von einem Orchestrator angewiesen. Ein Sub Agent erhält einen eingegrenzten Kontext, erledigt seine Arbeit und liefert ein Ergebnis zurück, ohne Bewusstsein vom übergeordneten Gesamtziel zu haben. Diese Isolation hält jeden Agent fokussiert und reduziert das Risiko, dass Fehler eines Agents das gesamte System beeinflussen.
