# Harness Engineering

> Das systematische Aufbauen der Infrastruktur, die ein Agent braucht, um sicher und effektiv auf einer Legacy-Codebasis arbeiten zu können: Test Harnesses, Build-Pipelines, Feedback-Loops, Tool-Integrationen und Kontextquellen. Sozusagen das Gerüst bauen, bevor der Agent loslegt. Ohne gutes Harness Engineering ist ein Agent blind und unkontrolliert, weshalb das Engineering der Leitplanken mindestens so wichtig ist wie der Agent selbst.
