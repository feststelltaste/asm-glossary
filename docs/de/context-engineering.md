# Context Engineering

> Die Praxis, die Informationen, die einem KI-Agenten bereitgestellt werden, gezielt zu entwerfen, zu strukturieren und auszuwählen, damit er innerhalb einer bestimmten Codebasis effektiv arbeiten kann. Geht über Prompt Engineering hinaus, das sich auf das Wie des Fragens konzentriert. Context Engineering fokussiert auf das Was: welches Wissen, welche Tools, welcher Speicher und welche Struktur dem Agenten zur Verfügung stehen, damit seine Ergebnisse mit Projektstandards, Richtlinien und Architekturabsichten übereinstimmen.
