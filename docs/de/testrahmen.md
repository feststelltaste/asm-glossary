# Testrahmen

> Automatisierte Testabdeckung, die vor der Modernisierung aufgebaut wird, damit der Agent nach jeder Änderung verifizieren kann, dass das Verhalten erhalten bleibt. Umfasst Unit-Tests, Integrationstests, Golden Master Tests und ggf. Contract Tests. Ohne Safety Net ist jede agentengestützte Codeänderung ein unkontrolliertes Experiment.
