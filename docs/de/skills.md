# Skills

> Umfangreichere, komplexere Agent-Verhaltensweisen, die durch bestimmte Schlüsselwörter oder Muster ausgelöst werden und aus mehreren Schritten, Tool-Aufrufen oder Sub Agents zusammengesetzt sind. Während Commands gespeicherte Prompts sind, sind Skills wiederverwendbare mehrstufige Workflows. Im Modernisierungskontext kann ein Skill den vollständigen Zyklus aus Analyse, Refactoring und Test eines Moduls kapseln.
