# Automatisiertes Refactoring

> Strukturelle Codeänderungen (Extract Method, Rename, Move Class), die ein Agent regelbasiert oder LLM-gestützt durchführt, im Unterschied zu manuellen IDE-Refactorings. Der Vorteil liegt in der Skalierung: Ein Agent kann hunderte gleichartige Refactorings konsistent über eine gesamte Codebasis ausführen. Voraussetzung ist ein belastbares Test Harness, das die Verhaltenserhaltung nach jeder Änderung sicherstellt.
