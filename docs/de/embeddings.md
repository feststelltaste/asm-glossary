# Embeddings

> Dichte numerische Vektorrepräsentationen von Text oder Code, die semantische Bedeutung erfassen. Ein Embedding-Modell wandelt einen Code-Ausschnitt, einen Dokumentationsabsatz oder eine Architekturbeschreibung in einen Vektor fester Größe um, sodass semantisch ähnliche Inhalte im Vektorraum nah beieinander liegen. Im Kontext der agentengestützten Modernisierung sind Embeddings die Grundlage von RAG: Das Retrieval-System des Agents findet relevanten Kontext durch Vektorvergleich statt Stichwortabgleich. Sie ermöglichen auch semantische Codesuche, also das Auffinden aller Stellen, die ein Konzept implementieren, unabhängig von Namenskonventionen, was keine grep-basierte Suche leisten kann.
