# Static Analysis

> Tools like SonarQube, PMD, or ArchUnit that an agent uses to check code quality, dependencies, and architectural violations. Provides the agent with objective metrics and structured information about the codebase. In the agentic loop, static analysis serves as a feedback channel that helps the agent evaluate the impact of its changes.
