# Agent Workflow Framework

> A runtime framework for defining and executing multi-agent pipelines as directed graphs, where nodes represent agents or tool invocations and edges define the flow of control and data between them. Provides structured state management across steps, enabling conditional branching, loops, and parallel execution paths. Where orchestration is the coordination concept, an agent workflow framework is the infrastructure that implements it, managing transitions, shared state, and error handling explicitly. Examples include LangGraph. Particularly relevant for modernization pipelines that require conditional routing based on analysis results.
