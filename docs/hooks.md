# Hooks

> Shell commands or scripts that an agent automatically executes at defined lifecycle points: before or after a tool call, on session start, or after a file edit. Enable automated enforcement of guardrails, formatting, logging, or validation without explicit agent instructions for each invocation. In the modernization context, hooks wire the agent into the existing build and quality pipeline.
