# Vibe Coding

> A development approach where engineers prompt an AI model conversationally, accept the generated code with minimal to no review, and iterate based on whether it works rather than whether it is understood. The term was coined by Andrej Karpathy in early 2025 and quickly entered mainstream usage. While effective for prototyping and exploration, Vibe Coding in production workflows accelerates both Dark Code and Cognitive Debt.
