# Characterization Test

> A test that records the current behavior of a legacy system without understanding it. Also known as "Golden Master Testing", this gives the agent a safety net even when no specification exists. The characterization test is created before modernization and serves as a reference: any change that alters the recorded behavior must be a conscious decision.
