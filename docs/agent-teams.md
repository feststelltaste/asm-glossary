# Agent Teams

> A coordinated group of sub agents working together on a shared goal. Each member handles a distinct concern: one agent may explore the codebase, another write tests, a third validate results. The team structure is defined by an orchestrator that assigns tasks, manages context handoff between agents, and aggregates their outputs. Agent teams enable parallelization and specialization that a single agent cannot achieve alone.
