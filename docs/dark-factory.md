# Dark Factory

> A fully autonomous software pipeline where AI agents write, test, and ship code without human involvement in the implementation. The term borrows from "lights-out" manufacturing, where factories run with no humans on the floor. Humans define business intent and validate outcomes but are intentionally excluded from authorship and review. A Dark Factory produces Dark Code by design, making robust test harnesses, fitness functions, and scenario-based validation essential safeguards.
