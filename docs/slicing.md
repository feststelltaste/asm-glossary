# Slicing

> Decomposing a system along its functional boundaries into thin, vertical slices that each represent a self-contained area of business capability. Each slice isolates a coherent part of the domain that can be modernized, tested, and deployed independently. Feature toggles can safeguard rollout of individual slices, and workload design determines whether each slice is handled by an agent, a human, or a combination. Slicing is particularly effective for agent-assisted work because a well-scoped slice fits within a single context window and can be completed in one agentic loop iteration.
