# Semantic Anchors

> Elements in code that carry meaning beyond their technical function and serve as signals for AI agents. Semantic anchors can be naming patterns, structural conventions, or domain vocabulary embedded in the code. They reduce ambiguity and give agents stable footholds for search, comprehension, and transformation.
