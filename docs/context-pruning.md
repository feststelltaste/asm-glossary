# Context Pruning

> Context Pruning is the deliberate removal of low-value, redundant, or off-topic content from an agent's context window to preserve space for meaningful signals. It is the proactive counterpart to truncation, which drops content passively when the window limit is reached, and complements compaction, which condenses context through summarization rather than removal. Common pruning targets include superseded instructions, completed task history, and unrelated file contents.
