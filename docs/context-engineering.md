# Context Engineering

> The practice of designing, structuring, and selecting the information provided to an AI agent so it can operate effectively within a specific codebase. Goes beyond prompt engineering, which focuses on how to ask. Context engineering focuses on what knowledge, tools, memory, and structure the agent receives, ensuring its output aligns with project standards, policies, and architectural intent.
