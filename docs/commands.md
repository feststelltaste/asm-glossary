# Commands

> Stored, reusable prompts that developers invoke as named commands, e.g., slash commands in tools like Claude Code. Turn frequently-used agent instructions into repeatable, shareable building blocks. In the modernization context, commands encode domain-specific workflows like "analyze this module for refactoring candidates" or "generate characterization tests for this class."
