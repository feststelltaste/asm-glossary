# Skills

> Larger, more complex agent behaviors triggered by specific keywords or patterns, composed of multiple steps, tool calls, or sub-agents. Commands are saved prompts; skills are reusable multi-step workflows. In the modernization context, a skill might encapsulate the full cycle of analyzing, refactoring, and testing a module.
