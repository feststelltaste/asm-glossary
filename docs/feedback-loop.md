# Feedback Loop

> A mechanism where the output of an agent is evaluated and the results are fed back to correct or refine subsequent actions. In agentic modernization, feedback loops connect code generation to test execution, static analysis, and human review, enabling the agent to self-correct rather than propagate errors across multiple transformation steps.
