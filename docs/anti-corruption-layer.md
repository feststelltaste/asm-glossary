# Anti-Corruption Layer (ACL)

> A translation layer between old and new systems that prevents legacy concepts from leaking into the new architecture. Protects the new domain logic from legacy baggage like outdated data models or inconsistent interfaces. Agents can generate and maintain ACLs because the translation rules are often well-formalizable.
