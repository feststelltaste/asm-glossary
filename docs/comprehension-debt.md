# Comprehension Debt

> The growing gap between the code that exists in a system and what any human actually understands. Unlike Cognitive Debt, which describes team knowledge eroding over time, Comprehension Debt is structural: AI-generated code enters the codebase faster than humans can review it, making full understanding physically impossible.
