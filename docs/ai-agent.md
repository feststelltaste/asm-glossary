# AI Agent

> An LLM-based system that autonomously executes multi-step tasks by planning, calling tools, and evaluating results, as opposed to pure prompt/response interaction. In the modernization context: the agent analyzes legacy code, derives refactoring steps, executes them, and verifies the outcome. Autonomy is deliberately constrained: an agent without guardrails is a risk, not a tool.
