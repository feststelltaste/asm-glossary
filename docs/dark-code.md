# Dark Code

> Dark Code is production code that no human has ever fully understood at any point in its lifecycle. Unlike legacy code that was once comprehended but lost institutional knowledge over time, Dark Code was generated, passed automated checks, and shipped without the comprehension step ever occurring. It emerges naturally in agentic workflows where authorship and understanding have decoupled.
