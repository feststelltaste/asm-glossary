# Pattern Diffing

> A change discipline in which modifications are crafted to produce structurally identical diffs across files, so that deviations can be spotted instantly. The reviewer's job shifts from understanding each change to verifying that the pattern holds and investigating any outlier, either by eye or by automated outlier detection over the diff shape. Turns volume into an advantage: the more uniform diffs there are, the easier it becomes to detect the one that does not belong.
