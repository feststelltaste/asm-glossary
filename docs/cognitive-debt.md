# Cognitive Debt

> The gradual erosion of shared system understanding within a team over time. Cognitive debt lives in people: it accumulates when developers accept code they don't fully understand, and when knowledge of how and why a system works is concentrated in too few minds. Particularly dangerous because it remains invisible until it manifests as inability to make decisions, fear of change, or unexpected system behavior. AI-generated code accelerates accumulation massively, because the learning effect of writing code yourself is eliminated.
